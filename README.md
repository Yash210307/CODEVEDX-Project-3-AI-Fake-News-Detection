# AI Fake News Detection Tool

AI Fake News Detection Tool is a Machine Learning and Natural Language Processing (NLP) based project designed to identify whether a news article is **Fake** or **Real**. The system analyzes the textual content of news articles, performs preprocessing and feature extraction, and uses a trained machine learning model to classify the news.

## Key Features

- Fake and Real news classification
- Text preprocessing and cleaning
- NLP-based feature extraction
- Machine Learning model training and prediction
- User-friendly interface for news verification
- Fast and accurate detection results

- Natural Language Processing (NLP)

- Misinformation detection
- Educational and research purposes
